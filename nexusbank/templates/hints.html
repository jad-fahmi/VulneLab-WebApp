NexusBank — Hints

Stuck? Here are some progressive hints for each flag.

Flag 1 — Login Page

Hint 1

Try logging in with unusual input.

Hint 2

What happens if you put a single quote in the username field?

Hint 3

Try: ' OR '1'='1 as the username with any password.

Flag 2 — Transfer Page

Hint 1

Try sending a transfer with something unusual in the comment field.

Hint 2

What if the comment contained HTML?

Hint 3

Try: <script>alert('XSS')</script> as the comment, then check the dashboard.

Flag 3 — Profile Page

Hint 1

Look closely at the URL when viewing your profile.

Hint 2

What happens if you change the number in the URL?

Hint 3

Try: /profile?id=2 or /profile?id=3.

Flag 4 — Admin Panel

Hint 1

Every banking app has an admin panel. Can you find it?

Hint 2

Try navigating to common admin paths.

Hint 3

Try: /admin — but think about how you might get in as a regular user.
